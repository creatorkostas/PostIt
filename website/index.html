PostIt is a powerful alternative to Postman, Insomnia, and Hoppscotch that runs entirely on your machine. No cloud, no accounts, no data leaving your network.

Built with Go for performance and distributed as a single binary, PostIt gives you a full-featured API development environment with zero dependencies.

Features

From REST to Kafka to load testing -- everything in one tool.

Request Builder

Full-featured HTTP client supporting GET, POST, PUT, DELETE, PATCH with raw JSON, form-data, GraphQL, and URL-encoded bodies. Headers with autocomplete, bearer token auth, and syntax-highlighted editors.

Variable System

Cascading variable resolution with local, environment, global, and OS-level scopes. Supports magic variables ($guid, $timestamp), interactive prompts, and O(1) environment lookups.

Script Engine

JavaScript-like pre-request and post-request scripts. Set and get variables with pm.collectionVariables, extract values from JSON responses, perform conditional logic, and chain requests.

Environments & Vault

Multiple environments (Local, Staging, Production) with secret variable encryption using AES-GCM and PBKDF2 with 100,000 iterations. Master password vault for sensitive data.

History & Time Travel

Full request/response recording with time-travel debugging. Export as JSON, CSV, or HAR format. Mermaid.js sequence diagram visualization of your request history.

Response Viewer

JSON syntax highlighting with expand/collapse, HTML preview in iframe, auto JWT decoding, side-by-side diffing against a pinned baseline, and an advanced JSON filter engine.

Quick Start

Get up and running in seconds.

01

Build

go build -o postit.exe ./cmd/postit
02

Run (Web UI)

./postit.exe --ui web --port 8080
03

Open

Navigate to http://localhost:8080 and start building requests.

Other Modes

./postit.exe --ui tui Terminal UI
./postit.exe --ui cli Interactive CLI
./postit.exe run --collection ./output/history.json Headless / CI/CD
